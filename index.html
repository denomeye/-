JSON이란?

JSON (JavaScript Object Notation)은 데이터를 저장하고 교환하는 텍스트 형식이에요.
사람이 읽기 쉽고, 프로그램도 파싱(읽기)하기 쉬워서 게임 데이터, API 통신 등에 널리 씁니다.

이 툴에서 내보낸 JSON 파일을 ChatGPT·Claude·Gemini 같은 AI에게 넘겨주면
"이 캐릭터 밸런스 검토해줘" "스킬 효과 보완해줘" 같은 작업이 가능해요.
